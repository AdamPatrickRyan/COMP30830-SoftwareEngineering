Disappster : Weather and Forecast Data

Created by Adam Ryan, last modified on Apr 15, 2021

This page details integration details (keys) of the application to OpenWeatherMapAPI to pull weather data.

The Table Below Lists the Integration Details and Endpoints.

The info returned is a json

For all integrations the appid parameter must be added to connect. Eg.

Open Weather Stations: https://api.openweathermap.org/data/2.5/weather?lat={}&lon={}&appid={API KEY GOES HERE}
