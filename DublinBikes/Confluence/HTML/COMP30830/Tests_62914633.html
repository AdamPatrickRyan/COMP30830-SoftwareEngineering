Disappster : Tests

Created by Adam Ryan, last modified on Apr 15, 2021

This page details the tests which were implemented. Due to time constraints and focusing on key features within the app, the tests are limited to just a database connection test. However, many of the methods contain detail error dictionaries and conditional statements designed to compensate for having failures.

One failing which would need to be fixed in a future iteration of the app is that we should have default data in the instance that the database connection fails to be established so that the app does not completely crash.
